    HTML



    Hypertext Makeup Language(HTML) is the standard markup language for creating web pages and web applications. Web browers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a Web page semantically and originally included cues for the appearance of the document.
